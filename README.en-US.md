## Preparing for use

Add the following dependencies to the `pubspec.yaml` file:

```dart
dependencies:
  bruno: version
```

## Simple usage

```dart
import 'package:bruno/bruno.dart';
...
BrnBubbleText(
  maxLines: 3,
  text: 'I am first bubble text',
)
```
